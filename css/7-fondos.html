7. Fondos

Propiedades de fondo en CSS

Disponemos de múltiples opciones de configuración de los fondos en CSS. Los fondos permiten mejorar la estética y la legibilidad de los elementos, y pueden combinarse para crear efectos visuales avanzados en cualquier parte de la página web.

Color de fondo

Para aplicar un color de fondo utilizamos la propiedad background-color. Se puede usar cualquier formato de color admitido por CSS: nombre estándar, hexadecimal, RGB, RGBA, HSL, HSLA, etc. Es importante elegir colores que garanticen buen contraste y accesibilidad para los usuarios.


    background-color: #000000;
    background-color: rgb(255,255,255);
    background-color: hsl(0, 0%, 100%);

Imagen de fondo

En un fondo también podemos poner una imagen, además de un color, con diferentes parámetros. Las imágenes de fondo pueden ser archivos locales o enlaces externos, y se recomienda optimizarlas para mejorar el rendimiento de la web. Es posible combinar color e imagen para lograr efectos de superposición.

  • Imagen: Indicamos la ruta donde se encuentra la imagen. La ruta tiene que ser relativa al archivo CSS o absoluta si es externa.
    background-image: url(ruta/imagen1.png);
    background-image: url('https://ejemplo.com/fondo.jpg');
  • Repetición: Indicamos si vamos a repetir la imagen en modo mosaico, en el eje X, en el Y, o no se repetirá. Usar no-repeat es útil para logos o imágenes decorativas, mientras que repeat es ideal para patrones.
    background-repeat: repeat; (mosaico)
    background-repeat: no-repeat; (no se repite)
    background-repeat: repeat-x; (eje X)
    background-repeat: repeat-y; (eje Y)
    background-repeat: space; (rellena con espacio)
    background-repeat: round; (escala para ajustarse)
  • Posición: Indicamos la posición de la imagen, bien con palabras clave o mediante coordenadas. Se pueden usar porcentajes, píxeles o palabras clave para adaptar la imagen al diseño.
    background-position: left top;
    background-position: center bottom;
    background-position: right center;
    background-position: 50% 20%;
    background-position: 25px 100px;
  • Comportamiento: Permite indicar el comportamiento de la imagen de fondo cuando se hace scroll. fixed mantiene la imagen fija en la ventana, creando el efecto "parallax".
    background-attachment: scroll; (no fija)
    background-attachment: fixed; (fija)

Forma abreviada

Todas estas propiedades se pueden escribir de forma abreviada en una sola línea, lo que facilita la gestión y el mantenimiento del código. El orden de los valores es importante: color, imagen, repetición, comportamiento y posición.


    background: #000 url(images/imagen1.png) no-repeat fixed left bottom;

Si solo usamos el color de fondo:


    background-color: #000; /* Forma larga */
    background: #000; /* Forma corta */

Si no queremos color de fondo, usamos transparent:


    background: transparent url(images/imagen1.png) no-repeat fixed left bottom;

Orden de las propiedades en background abreviado

En la forma abreviada de background, el orden de los valores es importante para que el navegador los interprete correctamente. El orden estándar recomendado es:

  1. Color de fondo
  2. Imagen de fondo (url)
  3. Repetición (repeat, no-repeat, repeat-x, repeat-y, space, round)
  4. Comportamiento (scroll, fixed)
  5. Posición (left top, center, right bottom, etc.)
  6. Tamaño (usando /, por ejemplo: center/cover)

No es obligatorio escribir todos los valores, pero este orden facilita la interpretación y el mantenimiento del código. Ejemplo completo:


    background: #ffcc00 url('fondo.png') no-repeat fixed center/cover;

Si omites algún valor, el navegador usará el valor por defecto para esa propiedad.

Otras propiedades

  • background-size: Ajusta el tamaño de la imagen. cover hace que la imagen cubra todo el elemento, aunque se recorte; contain ajusta la imagen para que se vea completa.
    background-size: 100%; (100% anchura, altura automática)
    background-size: 100% 50%; (100% anchura, 50% altura)
    background-size: cover; (cubre el bloque)
    background-size: contain; (se ve toda la imagen)
  • background-clip: Especifica si el fondo se extiende por debajo del borde, margen interno o contenido. text permite aplicar fondos a texto, creando efectos visuales avanzados.
    background-clip: border-box;
    background-clip: padding-box;
    background-clip: content-box;
    background-clip: text;
    color: transparent; (necesario para texto)
  • background-origin: Especifica el área de origen del fondo para calcular la posición de inicio. Permite controlar si la imagen se posiciona desde el borde, el padding o el contenido.

Múltiples imágenes de fondo

Para utilizar varias imágenes de fondo en el mismo elemento, se separan por comas. Es posible crear composiciones visuales complejas, como marcos decorativos o efectos de superposición. Recuerda que solo se puede poner un color de fondo, por lo que si se quiere añadirlo, se usa background-color aparte.


    blockquote {
      background:
        url(quote_open.gif) no-repeat top left,
        url(quote_close.gif) no-repeat bottom right;
      background-color: #CCC;
    }

Las imágenes de fondo no afectan el contenido ni la estructura del documento, pero pueden influir en la legibilidad y la experiencia visual. Es recomendable probar los fondos en diferentes dispositivos y resoluciones para asegurar una buena presentación.

Prioridad entre color e imagen de fondo

Cuando se utiliza la forma abreviada background: color url(), el color de fondo se aplica primero y la imagen se dibuja encima. Por lo tanto, la imagen tiene prioridad visual sobre el color. El color solo será visible en las siguientes situaciones:

  • Si la imagen tiene zonas transparentes (por ejemplo, formato PNG con transparencia), el color se verá en esas áreas.
  • Si la imagen es más pequeña que el área del elemento y no se repite, el color será visible en las partes no cubiertas por la imagen.
  • Si se usan varias imágenes de fondo con transparencias, el color puede aparecer debajo de todas ellas.

En la práctica, el color de fondo actúa como "base" y la imagen como "capa superior". Esto es útil para asegurar que el fondo nunca quede vacío si la imagen tarda en cargar, falla o no cubre todo el espacio.

Ejemplo visual:

background: #ffcc00 url('logo.png') no-repeat center;

Si logo.png tiene transparencia, el color amarillo #ffcc00 será visible en el fondo. Si la imagen cubre todo el área y es opaca, el color no se verá.
