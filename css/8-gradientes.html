8. Gradientes

Podemos generar colores con gradientes, sin necesidad de utilizar imágenes, configurando múltiples combinaciones de colores, tamaños, direcciones... Para ello utilizaremos un generador automático la mayoría de las veces.

Los gradientes en CSS se definen principalmente con las funciones linear-gradient(), radial-gradient() y conic-gradient(). Permiten crear transiciones suaves entre dos o más colores, y pueden aplicarse como fondo a cualquier elemento.

Gradiente lineal

Para crear un gradiente lineal establecemos un punto de partida y una dirección (indicada mediante un ángulo) a lo largo de la cual se aplicará el efecto. Es decir, la dirección del degradado lineal es una línea recta, que puede aplicar en cualquier ángulo.

También se pueden definir paradas de color e indicar si la transición entre estos colores será más o menos suave.

Ejemplo de sintaxis básica:
background: linear-gradient(90deg, #ff0000, #0000ff);

Se pueden añadir más colores y posiciones:
background: linear-gradient(45deg, #ff0000 0%, #00ff00 50%, #0000ff 100%);

Gradiente radial

Este tipo de gradiente nos permite crear degradados con formas circulares, ya que el gradiente radial se extiende hacia afuera en forma elíptica desde el punto central del círculo.

Ejemplo de sintaxis básica:
background: radial-gradient(circle, #ff0000, #0000ff);

Se pueden definir el centro, el tamaño y más colores:
background: radial-gradient(ellipse at center, #ff0000 0%, #00ff00 70%, #0000ff 100%);

Gradiente cónico

En este tipo de gradiente el punto de partida es el centro del gráfico y luego gira en el sentido de las agujas del reloj alrededor del centro para lograr un efecto de degradado.

Este tipo de gradiente cónico es muy reciente y hasta el año 2020 no hemos tenido un soporte generalizado del mismo.

Ejemplo de sintaxis básica:
background: conic-gradient(from 0deg, #ff0000, #0000ff);

Se pueden definir ángulos de inicio y múltiples colores:
background: conic-gradient(from 90deg, #ff0000 0deg, #00ff00 120deg, #0000ff 240deg);

Patrones para fondos

Combinando múltiples fondos y gradientes repetidos (y mucha imaginación y tiempo libre), podemos llevar los gradientes a otro nivel, consiguiendo originales fondos a modo de patrón, sin necesidad de utilizar archivos de imagen.

Para crear patrones complejos, se pueden superponer varios gradientes usando la propiedad background con múltiples valores separados por comas. Ejemplo:
background: linear-gradient(90deg, #fff 25%, transparent 25%), linear-gradient(180deg, #fff 25%, transparent 25%);
