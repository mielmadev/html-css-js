9. Tamaños de texto

Podemos establecer el tamaño de un texto usando diferentes unidades de medida. Estas unidades se utilizan también en otras propiedades CSS como los bordes, separaciones, anchuras, alturas, etc.

Nota: En HTML puedes usar varias clases en una misma etiqueta separándolas por espacios. Por ejemplo:
<p class="texto numero color">Ejemplo de párrafo</p>
Esto permite aplicar estilos de diferentes clases al mismo elemento.

Las unidades de medida se dividen en unidades absolutas y relativas:

Unidades absolutas

  • Pixels (px)
  • Puntos (pt)
  • Picas (pc)
  • Pulgadas (in)
  • Centímetros (cm)
  • Milímetros (mm)

En CSS, la unidad más utilizada es el px. No se recomienda usar unidades absolutas para el tamaño de texto por motivos de Responsive Web Design (RWD) y accesibilidad.

Unidades relativas al tamaño de la fuente

  • Porcentaje (%): relativo al tamaño del elemento padre.
  • em: relativo al font-size del padre.
  • rem: relativo al font-size del elemento raíz (html).

Ejemplo de equivalencias con tamaño base de 16px:

html { font-size: 16px; }

% 50% 55% 62.5% 75% 87.5% 100% 112.5% 125% 150% 200%

pt 6pt 7pt 7.5pt 9pt 10.5pt 12pt 13.5pt 14.5pt 18pt 24pt

Para facilitar equivalencias, se puede usar un tamaño base de 10px:

html { font-size: 10px; }

Con este tamaño base, 1em = 10px, 1.5em = 15px, 2em = 20px, etc.

Si se usa un tamaño base de 62.5% (10px sobre 16px), las equivalencias serían:

Unidades relativas al viewport

  • vw: ancho del viewport
  • vh: alto del viewport
  • vmin: mínimo entre ancho y alto
  • vmax: máximo entre ancho y alto

Al explorador le cuesta calcular estas unidades, por lo que pueden no ser precisas en todos los casos y no es recomendable usarlas en exceso.

Ejemplo: 1vw = 1% del ancho del navegador

No se recomienda abusar de estas unidades en textos pequeños.

Función clamp() en CSS

La función clamp() permite definir valores que se mantienen dentro de un rango mínimo y máximo, ideal para tamaños de texto fluidos y responsivos.

font-size: clamp(valor-mínimo, valor-preferido, valor-máximo);
  • valor-mínimo: el tamaño más pequeño permitido.
  • valor-preferido: el valor ideal, normalmente una unidad relativa o calculada.
  • valor-máximo: el tamaño más grande permitido.

Ejemplo práctico:

h1 {
                font-size: clamp(1.5rem, 4vw, 3rem);
              }

En este ejemplo, el tamaño de la fuente será como mínimo 1.5rem, idealmente 4vw (fluido según el ancho de la ventana), y como máximo 3rem.

Ventajas:

  • Permite tipografía responsiva sin media queries.
  • Mejora la accesibilidad y el diseño adaptable.

Más información: MDN Web Docs: clamp()

Nota: Para mejorar la accesibilidad y el diseño adaptable, utiliza siempre unidades relativas (em, rem, %) en los textos. Las unidades absolutas pueden dificultar la visualización en diferentes dispositivos y tamaños de pantalla.
