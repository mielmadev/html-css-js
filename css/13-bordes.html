13. Bordes

Bordes básicos

Sobre un elemento HTML podemos aplicar un borde, el cual puede ser idéntico en los 4 lados o bien tener bordes diferentes.

Un borde se compone de 3 elementos: grosor, estilo de la línea y color, separados por un espacio. Es igual que las propiedades de decoración que hemos visto anteriormente. El orden en que se indiquen estos 3 elementos es irrelevante.

Vamos a ver primero las propiedades por separado y luego las abreviaremos.

Grosor:

Establecemos el grosor del borde. No se pueden utilizar porcentajes para establecer el grosor.

border-width: 20px; border-width: 10px;

Estilo

Disponemos de diferentes estilos de borde:

border: solid; border: dotted; border: dashed; border: double; border: inset; border: outset; border: groove; border: ridge; border: none;

Color:

Utilizamos un color con el formato que queramos (RGB, Hexadecimal, etc.).

border-color: #FF0000; border-color: #00FF00; border-color: #0000FF; border-color: #00FFFF; border-color: #FFFF00; border-color: #000000;

Combinando las 3 opciones, obtenemos este resultado:

border-width: 0px
border-width
border-style:
  • None
  • Solid
  • Dotted
  • Dashed
  • Double
  • Inset
  • Outset
  • Groove
  • Ridge
border-color: Transparent Red Green Blue Yellow Black

Hasta el infinito y más allá

Si tenemos 4 lados y 3 propiedades por lado, en total tendríamos que utilizar 12 propiedades para establecer un borde completo.

Vamos a abreviar las 3 propiedes en una única, border:

border: grosor estilo color; (o el orden que queramos, es irrelevante)
border: 20px solid #FF0000;

Si queremos dar estilo a cada lado de forma individual, llamaremos a las propiedades de cada lado:

border-top: 20px solid #FF0000; /* borde superior */
border-right: 10px dashed #00FF00; /* borde derecho */
border-bottom: 5px double #0000FF; /* borde inferior */
border-left: 3px dotted #00FFFF; /* borde izquierdo */

Bordes multicolor

Para que el borde double funcione correctamente, será necesario utilizar un grosor con número impar.

Para visualizar el borde inset | outset | groove | ridge, necesitaremos los 4 bordes visibles.

Bordes redondeados

Podemos cambiar la apariencia de nuestros bordes, de cuadrados a redondeados.

Se puede configurar el radio de la curvatura con el mismo valor para todos los lados del borde, o bien configurar uno diferente para cada lado, de manera individual:

Todas las esquinas iguales:

border-radius

Esquinas individuales:

border-top-left-radius
border-top-right-radius
border-bottom-right-radius
border-bottom-left-radius

No hace falta escribir las propiedades de forma individual. Podemos unificar las propiedades. En este caso utilizaremos 4 parámetros separados por espacios y el orden debe ser en el sentido de las agujas de reloj, empezando por esquina superior izquierda:

border-radius: 10px 20px 30px 60px;

Bordes

Esquinas simétricas

Si tenemos esquinas simétricas, podemos escribir tan solo 2 valores:

border-radius: 50px 20px;

Bordes

En este caso, el primer valor (50px) se aplicará en la esquina superior izquierda y en la inferior derecha, y el segundo valor (20px) se aplicará en la esquina superior derecha y en la inferior izquierda.

Elementos redondos

Podemos hacer que un bloque de contenido o una imagen tenga forma de círculo perfecto simplemente poniendo un borde redondeado igual para las cuatro esquinas y cuyo tamaño sea el 50% de cada lado:

/* cuadrado (el general): */
img{
  border: 2px solid #000;
}

/* redondo: */
img{
  border: 2px solid #000;
  border-radius: 50%;
}

Bordes

Para que este efecto se visualice bien, es necesario que la imagen o el bloque tengan un tamaño cuadrado (misma anchura y altura).

Otros bordes

Existe la posibilidad de crear bordes con imágenes, gradientes y bordes animados, combinando más propiedades y técnicas.
