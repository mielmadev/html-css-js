2. Validación

Al igual que hemos hecho con el código HTML, el código CSS lo tendremos que validar para evitar errores y malas implementaciones.

En este caso es de vital importancia porque si hay un código erróneo, puede que no se visualice la página o que no se aplique el formato visual que queramos.

Utilizaremos el validador de código CSS que el W3C pone a nuestra disposición. Y también lo podemos validar de 3 maneras diferentes:

  • Indicando la dirección web de una página que sea pública.
  • El validador busca las CSS enlazadas, embebidas...
  • Subiendo nuestro archivo css local
  • Copiando y pegando todo el código fuente
